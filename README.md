# Student Performance Analysis using Power BI

This project analyzes student academic performance using Power BI.

It studies the effect of:
- Parental education
- Attendance
- Previous scores
- Extracurricular activities

## Tools Used
Power BI, Excel, DAX

## Key Insights
- Students with higher previous scores perform better
- Parental education impacts performance
- Extracurricular activities improve results
